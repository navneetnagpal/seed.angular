
# Seed project for angular 
A project to kick start
# Pre-requisites 
- nodemon should be installed globally
- npm install -g nodemon
- gulp should be installed globally
- npm install -g gulp

# Steps to run
- npm install
- gulp build:all
- npm start


# Build Tasks
- gulp scripts.core
- gulp scripts.core:watch
- gulp scripts.app
- gulp scripts.app:watch
- gulp scripts:all
- gulp styles


# Tests
- gulp test
